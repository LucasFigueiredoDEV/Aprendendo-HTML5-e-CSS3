Links e Âncoras

Links com redirecionamento, mas sem volta ao site.

Para criar um link é preciso criar uma âncora em alguma imagem, botão ou palavra.

É recomendado que caso você queira que o usuário seja redirecionado à outra página, mas que não volte para a sua só seja criado uma âncora normal.
